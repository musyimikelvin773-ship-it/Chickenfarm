# Paul's Chicken Farm

This is a small static site for Paul's Chicken Farm. I refreshed the homepage with a modern layout, responsive CSS, product cards, sample images (from Unsplash), and an inquiry modal that opens your mail client.

How to preview locally

1. Open `index.html` in your browser (double-click or use "Open with Live Server" in VS Code).
2. The inquiry buttons open a modal and then launch your mail client with a prefilled message when you submit.

Notes
- Images use Unsplash "source" image URLs and may vary over time.
- To publish on GitHub Pages, enable Pages in the repository settings and set the branch to `main` (or `gh-pages`).
